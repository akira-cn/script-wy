# 腳本秘術

## 安裝

用如下命令之一安裝至藏書樓。

```
wyg i 腳本秘術
wyg i script
```

在文言代碼中嵌入多行JS腳本。

## 用法示例

```
吾嘗觀「「腳本秘術」」之書。

爪哇國有腳本
function f1() {
  return "f1";
}
function f2() {
  return "f2";
}
其文如是矣。

施「f1」。書之。
施「f2」。書之。
```

## 注意

💡限制：在嵌入的JS代碼裡任何位置不可出現任何文言的關鍵字，包括字符串、註釋。

所以在JS代碼裡儘量不要使用中文。

例如：

```
爪哇國有腳本
function say() {
  console.log('文言');
}
其文如是矣。
```

上面這段代碼會編譯錯，因為“言”是關鍵字。
